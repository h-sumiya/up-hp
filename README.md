# Unbiased Programming Website

## Description
上智大学のプログラミングサークル「Unbiased Programming」の公式ウェブサイトです。

## Development

### Clone
```bash
git clone git@github.com:h-sumiya/up-hp.git
cd up-hp
```

### Initialize
```bash
npm install
```

### Run
```bash
npm run dev
```

### Build
```bash
npm run build
```
